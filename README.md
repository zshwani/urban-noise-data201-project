# Data 201 First Project – Urban Noise Levels

This repository contains my Data 201 first project completed in Google Colab.

## Project Summary
The dataset (`urban_noise_levels.csv`) contains 2,000 simulated measurements of environmental noise recorded in different urban conditions.  
The Jupyter notebook (`Urban_Noise_Data201_Project.ipynb`) includes:

1. **Introduction:** Overview of dataset, variables, and research questions.
2. **Data Wrangling:** Cleaning and feature engineering using Pandas and NumPy.
3. **Exploratory Data Analysis:** Histograms, boxplots, scatterplots, and pairplots using Matplotlib and Seaborn.
4. **Nonparametric Inference:** Bootstrapped confidence interval for the median noise level.
5. **Results Summary:** Interpretation of visualizations and statistical findings.
